JavaScript Variables

What is a Variable?

A variable is a container used to store data in JavaScript. It allows programmers to store values and reuse them later in the program.

Example

let name = "Aadil";
let age = 26;

console.log(name);
console.log(age);

Explanation

  • let creates a variable
  • name stores text
  • age stores a number
  • console.log() prints the value
