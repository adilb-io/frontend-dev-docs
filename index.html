JavaScript Booleans

JavaScript Booleans

What is a Boolean?

A Boolean represents only two values: true or false.

Example

let isLoggedIn = true;
let isStudent = false;

console.log(isLoggedIn);
console.log(isStudent);

Explanation

  • Boolean values are true or false
  • They are used in conditions
  • Important for decision making
JavaScript Variables

JavaScript Variables

What is a Variable?

A variable is a container used to store data in JavaScript. It allows programmers to store values and reuse them later in the program.

Example

let name = "Aadil";
let age = 26;

console.log(name);
console.log(age);

Explanation

  • let creates a variable
  • name stores text
  • age stores a number
  • console.log() prints the value
JavaScript Arrays

JavaScript Arrays

What is an Array?

An array stores multiple values in a single variable. It is useful when working with lists of data.

Example

let fruits = ["apple","banana","mango"];

console.log(fruits);
console.log(fruits[0]);

Explanation

  • Arrays store multiple items
  • Items are separated by commas
  • Index starts at 0
JavaScript Objects

JavaScript Objects

What is an Object?

Objects store related data together using key and value pairs.

Example

let student = {
 name: "Aadil",
 age: 26,
 city: "Hyderabad"
};

console.log(student.name);

Explanation

  • Objects use { }
  • Data is stored as key:value
  • Values can be accessed using dot notation
